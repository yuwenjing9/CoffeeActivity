## 页面结构说明

### 顶部导航栏
- 背景色：`#6B4423`（咖啡棕）
- 标题："鲜萃咖小站"
- 副标题："每一杯都是精心之作"
- 右侧咖啡图标：☕

### Banner轮播
- 高度：336rpx
- 自动轮播，间隔3秒
- 底部指示器

### 快捷分类
- 4个分类入口
- 圆形图标背景（`#FAF8F5`）
- 分类名称：美式咖啡、拿铁系列、卡布奇诺、冰咖啡

### 限时优惠
- 渐变背景（焦糖橙到咖啡棕）
- 显示"限时优惠"和"全场满50减10"

### 推荐商品
- 2列网格布局
- 每个商品卡片包含：
  - 商品图片
  - 标签（热销/推荐/新品）
  - 商品名称
  - 价格
  - 评分

### 底部TabBar
- 4个Tab：首页、购物车、订单、我的
- 首页为激活状态，颜色`#6B4423`
- 其他Tab未激活状态，颜色`#999999`

## 设计规范

### 颜色
- 主色：`#6B4423`（咖啡棕）
- 辅助色：`#C17817`（焦糖橙）
- 推荐标签：`#7A9E7E`（抹茶绿）
- 背景色：`#FAF8F5`（米白）
- 文字色：`#333333`、`#666666`、`#999999`

### 字体大小
- 标题：31.5rpx (15.75px)
- 正文：24.5rpx (12.25px)
- 小字：21rpx (10.5px)

### 间距
- 页面左右边距：35rpx
- 卡片间距：21rpx
- 圆角：29rpx

## 使用说明

所有资源文件已复制到 `xclaw-cafe/miniprogram/images/` 目录下，可直接通过相对路径引用。

示例：
```wxml
<image src="/images/15.png" />
<image src="/images/11.svg" />
```
